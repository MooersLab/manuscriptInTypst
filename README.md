# First submission manuscript template in typst

## Purpose

Prepare PDF for initial submission of a manuscript.

## Features

- line numbering
- single- and double-spacing
- short author list and running title in header
- 1/N page numbering
- biblatex
- example
  + figure
  + table
  + equation
